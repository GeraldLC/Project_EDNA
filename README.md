# Jenisa Carvalho — Interior Design Portfolio

> A minimal, aesthetic portfolio showcasing thoughtful interior spaces.  
> Built with GitHub Pages · Custom domain coming soon.

---

## Overview

This is the source code for **Jenisa Carvalho's personal interior design portfolio** — a clean, visually driven site designed to let the work speak for itself. Jenisa is a London-based Interior Designer and MA graduate (First Class) from Middlesex University, with hands-on experience spanning residential, hospitality, and luxury kitchen interiors. The site focuses on elegant presentation with a minimalist aesthetic, putting the spotlight on spaces, materials, and mood.

## Setup & Development

1. **Clone the repo**
   ```bash
   git clone https://github.com/GeraldLC/Project_EDNA.git
   cd Project_EDNA
   ```

2. **Open locally**  
   Just open `index.html` in your browser — no build step required.

3. **Deploy**  
   Push to the `main` branch. GitHub Pages auto-deploys from root or `/docs` folder depending on your settings.

---

## Custom Domain (Coming Soon)

To connect a custom domain later:
1. Add a `CNAME` file to the repo root with your domain (e.g. `jenisacarvalho.com`)
2. Update DNS records with your domain registrar
3. Enable HTTPS in GitHub Pages settings
